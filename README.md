# Algorithm
알고리즘 문제 풀이를 저장하는 곳입니다.

## 커밋 규칙
***2021.9.17부터 적용**
### Format

#### {Keyword}: [{#Issue}] {출처} {난이도} {제목}
- 레포지토리 내 Issues에 문제와 관련한 자료구조/알고리즘 이슈 추가
- 문제 Commit 시 관련 이슈 넘버 태그 

### Keyword
- `solve`: 풀이에 성공한 새 문제 추가
- `fail`: 풀이에 실패한 새 문제 추가
- `improve`: 새로운 성공 풀이를 추가한 문제
- `docs`: readme 등 문서 수정
